**Obiettivo:** Scrivere un programma che permette all'utente di scegliere un prodotto da una lista e inserire la quantità, gestendo gli errori senza far bloccare il programma.

### Scenario

Hai un array di prodotti (es. Snack, Bibita, Caffè).  
L'utente inserisce il numero del prodotto e la quantità.  
Il programma deve calcolare il prezzo totale.

### Requisiti Tecnici

Il programma deve gestire **due** casi usando `try` e `catch` (o multi-catch):

1. **L'utente inserisce testo invece di un numero**
    - *Eccezione:* `java.util.InputMismatchException`

2. **L'utente sceglie un numero di prodotto inesistente**
    - *Eccezione:* `ArrayIndexOutOfBoundsException`
